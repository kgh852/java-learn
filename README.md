# class

```java
class person {
    int age = 17;
    String name = "김구현";
    String school = "경소고";
}

public class Main {
    public static void main(String[] args) {
        person guhyun = new person();
        System.out.println(guhyun.age);
        System.out.println(guhyun.name);
        System.out.println(guhyun.school);
    }
}
```
